# Tresor React Assignment

## For Backend

### Clone The Repo and go to Backend Dir : 

```
git clone https://github.com/vaibhavydv/tresor-react-assignment
```
### Install NPM Packages and Start Development Server 

``` bash
npm install 

npm start
```
